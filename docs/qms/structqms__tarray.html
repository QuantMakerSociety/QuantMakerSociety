Detailed Description

template<typename T>
struct qms_tarray< T >

Array

Template Parameters
TType of item.

Can be thought as a vector of pointer to items.

Warning
It DOES NOT OWN its items. This is done by internal design.
it does not have a destructor. Again done by internal design.
